Emotional Support Pickle (ESP)

Emotional Support Pickle (ESP) is a decentralized meme token with on-chain reward mechanics. ESP combines playful branding with utility by distributing variable rewards sourced from real trading activity.

ESP does not rely on minting, rebasing, or inflationary emissions.

Key Features
	•	Fixed supply
	•	Reflection-style rewards
	•	No staking or lockups
	•	Non-custodial
	•	Community-driven
